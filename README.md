# Student Grade Tracker

## Overview
The Student Grade Tracker is a Python project that reads student grades from a CSV file, calculates averages, assigns letter grades, and creates a class performance summary.

## Features
- Reads student data from a CSV file
- Calculates student averages
- Assigns letter grades
- Calculates class average
- Identifies highest and lowest performing students
- Displays a clear report in the terminal

## Technologies Used
- Python
- CSV
- Functions
- Loops
- Lists and dictionaries

## Skills Demonstrated
- Data processing
- Data analysis basics
- File handling
- Reporting
- Problem-solving
- Clean code organization

## How to Run
1. Download the project files.
2. Make sure `main.py` and `grades.csv` are in the same folder.
3. Run:

python main.py
